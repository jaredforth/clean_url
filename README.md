# clean_url

This crate takes a URL, formats it correctly, and returns the version of the URL that returns a 200 series response. 

Documentation:
-   [API Reference](https://docs.rs/clean_url)

## Installation and Usage

### Installation 

Add this to your `Cargo.toml`:

```toml
[dependencies]
clean_url = "0.1"
```
